# Bandanize Frontend

Modern React application for Bandanize, the collaborative platform for musicians.

## Overview

A responsive, dark-themed web application that allows musicians to:
- Manage musical projects and bands.
- Organize setlists and songs.
- Edit tablatures with a built-in specialized editor.
- Upload and manage media assets (Audio, Images).
- see real-time member presence in projects.

## Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: TailwindCSS
- **UI Components**: Shadcn/UI (Radix Primitives)
- **Icons**: Lucide React
- **State Management**: React Context API
- **Routing**: React Router DOM

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Key Features

- **Project Hub**: Central dashboard for your band.
- **Tab Editor**: Interactive editor for Guitar/Bass tabs with playback support.
- **Responsive Design**: Fully optimized for Desktop and Mobile.
- **Real-time Updates**: "Online" status indicator for project members.
- **Dark Mode**: Sleek, contrast-optimized dark theme.
